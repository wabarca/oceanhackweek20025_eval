Monitoreo y pronstico de clorofila-a en zonas costeras utilizando datos del servicio Copernicus Marine Environment Monitoring Service (CMEMS)

Este cdigo permite monitorear y pronosticar la concentracin de clorofila-a en zonas costeras utilizando datos satelitales proporcionados por el servicio Copernicus Marine Environment Monitoring Service (CMEMS).

El catlogo de diferentes datasets est disponible en: https://marine.copernicus.eu/ y para poder acceder y descargar los datos es necesario registrarse gratuitamente en la plataforma.

Para acceder a los datos se utiliza Copernicus Marine Toolbox con su API en python, medienta la libreria copernicusmarine. Adems se utilizan bibliotecas como xarray, matplotlib y cartopy para la manipulacin de datos y la visualizacin geoespacial.

Importante: la libreria copernicusmarine permite hacer login interactivo y tambin automatizado. En este caso se utiliza el login automatizado mediante un archivo de configuracin de credenciales, si no se tiene es necesario crearlo en el directorio de trabajo, la misma libreria permite generarlo:

import copernicusmarine
copernicusmarine.login()

Descargamos los datos de clorofila-a satelitales y de pronstico para una regin costera especfica.

Los datasets utilizados son:

  • Observaciones: Global Ocean Colour (Copernicus-GlobColour), Bio-Geo-Chemical, L4 (monthly and interpolated) from Satellite Observations (Near Real Time)
  • Pronstico: Global Ocean Biogeochemistry Analysis and Forecast.

Se seleccionan las ltimas 3 fechas disponibles para observacin (da anterior al actual es la ms reciente) y 3 tiempos de pronstico (el dia actual es el primer tiempo).

Se ingresan las coordenadas geogrficas de la regin costera de inters, adems de un nombre descriptivo para la misma (evitar espacios).

Se descargan los datasets en formato netCDF y se almacenan en un directorio local para su posterior anlisis y visualizacin. Los archivos se renombran de acuerdo a la fecha actual en el formato YYYYMMDD_chlo_fcst.nc (pronstico) y YYYYMMDD_chlo_obs.nc (observacin).

Se generan los grficos comparativos de concentracin de clorofila-a para las fechas seleccionadas, mostrando tanto las observaciones satelitales como los pronsticos del modelo. Los grficos incluyen mapas geoespaciales con proyecciones cartogrficas adecuadas y barras de color para interpretar los niveles de clorofila-a.
